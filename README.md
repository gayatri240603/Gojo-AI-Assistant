# Gojo AI Assistant

An AI-powered voice assistant inspired by Satoru Gojo from Jujutsu Kaisen.

## Features
- Voice Commands
- Offline AI Chat using Ollama
- TinyLlama Integration
- Speech Recognition
- Edge-TTS Voice Output
- Gojo-inspired GUI
- Open Websites & Execute Commands

## Tech Stack
- Python
- Ollama
- TinyLlama
- CustomTkinter
- SpeechRecognition
- Edge-TTS
- Pygame

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Gojo-AI-Assistant.git
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Install and run Ollama:

```bash
ollama run tinyllama
```

Run the application:

```bash
python gui.py
```
